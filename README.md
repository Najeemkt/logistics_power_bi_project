# logistics_power_bi_project

Logistics Performance & Sales Analysis
Objective

The main objective of this project is to analyze logistics and supply chain data to gain insights into sales performance, delivery efficiency, geographical trends, and customer behavior.
The dashboard helps stakeholders understand operational performance, identify delays and inefficiencies, and support data-driven decision-making to improve overall business outcomes.

Dataset

Source: Synthetic dataset prepared for academic analysis
Number of rows: ~75,000 records
Number of columns: 22 attributes (after cleaning & normalization)

Columns:
order_id, order_date, delivery_date, customer_id, product_id, region, country, shipment_mode, carrier, order_status, quantity, revenue, profit, shipping_cost, delivery_days, delivery_status, customer_segment, product_category, sub_category, rating, expected_delivery_date, delay_flag
